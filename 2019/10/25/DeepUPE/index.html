Underexposed Photo Enhancement Using Deep Illumination Estimation

ABSTRACT

—

  • This paper presents a new neural network for enhancing underexposed photos. Instead of directly learning an image-to-image mapping as previous work, we introduce intermediate illumination in our network to associate the input with expected enhancement result, which augments the network’s capability to learn complex photographic adjustment from expert-retouched input/output image pairs.

CONTRIBUTION

—

  1. We propose a network for enhancing underexposed photos by estimating an image-to-illumination mapping, and design a new loss function based on various illumination constraints and priors.
  2. We prepare a new dataset of 3,000 underexposed images, each with an expert-retouched reference.
  3. We perform evaluation on our method using existing and new datasets, and demonstrate the superiority of our method qualitatively and quantitatively.
